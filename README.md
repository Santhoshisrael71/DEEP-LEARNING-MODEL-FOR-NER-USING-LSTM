# DEEP-LEARNING-MODEL-FOR-NER-USING-LSTM

## Project Overview

This project implements Named Entity Recognition (NER) using a Deep Learning model in PyTorch.
The model identifies entities such as Person, Location, and Organization from text.

## Technologies Used

Python

PyTorch

NumPy

Pandas

Scikit-learn

Jupyter Notebook

## Project Files

NER_Exp.ipynb – Main notebook containing the model

Dataset files – Training and testing data

README.md – Project description

Workflow

Load and preprocess the dataset

Convert text into numerical format

Train the deep learning model

Evaluate model performance

Predict named entities from text
